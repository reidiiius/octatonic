# Octatonic
Hexadecimal fretboard diagrams

---

### Download

    git clone https://github.com/reidiiius/octatonic.git

---

### Usage
command line

    cd octatonic

    js -s korsakov.js | less

    js -s korsakov.js cgdae | less

    js -s korsakov.js bfbfb | less
